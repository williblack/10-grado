Prof. William Fdo Bernal

🎯 Objetivo de Aprendizaje

Identificar los componentes de un circuito eléctrico y comprender el flujo de energía en circuitos abiertos y cerrados.

Elementos del Circuito

🔋

1. La Fuente

(Pasa el ratón)

El Generador

Proporciona el "empuje" o la energía necesaria para que los electrones se muevan por el circuito.

🧵

2. El Conductor

(Pasa el ratón)

La Autopista

Material (como el cable de cobre) que permite el paso fácil de la electricidad.

💡

3. El Receptor

(Pasa el ratón)

El Trabajador

Recibe la energía y la transforma en luz, calor o movimiento (ej. un bombillo).

🎚️

4. El Interruptor

(Pasa el ratón)

El Puente

Controla el circuito. Si se levanta (abierto), corta la energía. Si baja (cerrado), deja pasar la energía.
